ARCH ROBOT LAB
Database of Robots

Installation

1. git clone https://github.com/iago84/ARCHROBOTLAB.git
2. cd ARCHROBOTLAB
3. activate venv
4. python manage.py createsuperuser
5. python manage.py runserver



Usage

1. Enter localhost /admin
2. Log as SuperUser
3. Fill Models and Fields as you wish

Common
1. Create ROBOT
2. Create Component
3. Create Code Library
4. Create Structure
